The project executes successfully only after all the required setup is done on AWS, as detailed below STEPS

--------------------------------------------------------------------------------------------------------------------
STEP 1).    From AWS IAM UI - Create IAM Roles (for EC2 and Lambda to be able to use other AWS services like Dynamodb, SNS, andKinesis)
--------------------------------------------------------------------------------------------------------------------
STEP 2).    Create EC2 instance and SSH using Putty (Name = yahoofinance)
--------------------------------------------------------------------------------------------------------------------
STEP 3).    Create Kinesis Data Stream (Stream Name = LoadStockPrice)
--------------------------------------------------------------------------------------------------------------------
STEP 4).    Create DynamoDB Table (Table Name = POIAlert)
--------------------------------------------------------------------------------------------------------------------
STEP 5).    Create SNS Topic (Topic Name = POIAlert)
--------------------------------------------------------------------------------------------------------------------
STEP 6).    Create SNS Subscription (Email-JSON = abc@outlook.com) //use any email address here
--------------------------------------------------------------------------------------------------------------------
STEP 7).    Create Lambda Function (Function Name = POIDetector - see POIDetectorLambdaFunction.py for Code)
--------------------------------------------------------------------------------------------------------------------
STEP 8).    Run StockPriceIngestion.py on EC2 Terminal
--------------------------------------------------------------------------------------------------------------------
STEP 9).   	Check Table created at Step 4 for Alert data records
--------------------------------------------------------------------------------------------------------------------
STEP 10).   Check Email provided at Step 6 for Alert emails with alert data records
--------------------------------------------------------------------------------------------------------------------
STEP 11).   Check "ACSC_OCT21_SEEMA_NAIR_YahooFinance_CloudProject1.docx" for required screenshots.
